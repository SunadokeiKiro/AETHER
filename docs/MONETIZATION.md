# AETHER マネタイズ戦略

## 概要

本ドキュメントはAETHERの収益モデルと課金体系を定義します。
「基本無料（Freemium）」をベースに、ユーザーのエンゲージメントに応じて課金ポイントを分散させます。

---

## 課金モデル

### ① サブスクリプション（AETHER Premium）

安定した収益の柱です。AIのトークン消費量が多い機能をここに集約します。

| 機能 | 説明 |
|------|------|
| **AIディレクターのフル解放** | メモからのタスク自動抽出、スケジュールの自動構成、音声での進捗フィードバック |
| **無制限のGenerative UI** | 自然言語によるUI変更回数の無制限化（無料版は1日3回） |
| **高度なコンテキスト認識** | 複数の標準機能を跨いだ複雑なAI操作 |
| **広告非表示** | ホーム画面や機能移行時の広告削除 |
| **クラウド同期** | 複数デバイス間でのデータ同期 |

---

### ② インアップ購入（IAP / 個別課金）

所有欲やパーソナライズに訴求します。

| アイテム | 説明 |
|----------|------|
| **プレミアム・デザインプリセット** | 著名デザイナーや特定のブランドとコラボした「究極のテーマ（トークンセット）」の販売 |
| **AIボイスパック** | AIディレクターの「声」や「性格」を、有名声優や特定のキャラクターに変更する権利 |
| **追加機能モジュール** | 専門的な計算機（金融計算、エンジニアリング用）などの高度な拡張機能の単体販売 |

---

### ③ リワード広告（ハイブリッドモデル）

非課金ユーザー向けの救済措置兼、収益源です。

| オプション | 説明 |
|------------|------|
| **AIトークンのチャージ** | 30秒の動画広告視聴で、UI変更やAI操作を1回分解放 |
| **期間限定プレミアム** | 広告視聴で1時間だけ全機能を解放 |

---

## 機能 × 課金形態 マトリクス

| 機能 | Free (無料) | Premium (月額) | One-time (個別) |
|------|-------------|----------------|-----------------|
| **標準ツール (電卓等)** | 全機能利用可 | 広告なし | - |
| **Generative UI** | 1日3回まで | 無制限 | 限定テーマ購入 |
| **AIディレクター** | 簡易タスク抽出 | 全自動連携 | - |
| **AIボイス/人格** | 標準のみ | 全種解放 | 特定ボイス購入 |
| **クラウド同期** | なし | あり | - |

## 価格戦略

### サブスクリプション料金

| プラン | 価格 | 備考 |
|--------|------|------|
| **月額プラン** | ¥500〜¥1,200 | 標準価格 |
| **年額プラン** | 月額の20%OFF | まとめ払い割引 |

### 損益分岐点の考慮

- Gemini APIコスト: 約$1.0 / 1M tokens
- ユーザーあたりの平均トークン消費量を算出
- 赤字にならない価格設定が必須

## 実装準備

### A. 技術的準備（Billing & API）

| 項目 | 詳細 |
|------|------|
| **Google Play Billing Library** | v6.0以降を使用。定期購読のステータス管理、購入の承認（Acknowledge）フロー実装 |
| **APIコスト予算管理** | ユーザーごとのGemini消費トークン量カウント。無料枠超過時の利用制限ダイアログと課金誘導 |
| **領収書検証 (RTDN)** | Firebase Functions等でのバックエンド構築。決済不正防止と複数デバイス間での課金状態同期 |

### B. 規約・法務的準備

| 項目 | 詳細 |
|------|------|
| **特定商取引法に基づく表記** | 日本国内向け販売の必須情報整理 |
| **プライバシーポリシー更新** | ユーザー入力データ（メモ・音声）のAI処理、課金情報の取り扱いを明記 |
| **Google Playポリシー遵守** | AI生成コンテンツの宣言提出準備、サブスク解約方法の明示 |

### C. ビジネス・マーケティング準備

| 項目 | 詳細 |
|------|------|
| **ASO用資産** | 「AIでUIが変わる瞬間」のプロモ動画、「標準機能統括」を強調したスクリーンショット |
| **価格テスト** | A/Bテストによる最適価格帯の検証 |
